Data Source

Our race results data is sourced from a Kaggle Dataset.

The dataset features three years of Boston Marathon Results data. Each year of data contained over 26,000 runners for a combined 79,000 rows of race results. The dataset population was about 45% female and 54% male.

We used several attributes to train and test the machine models: bib number, gender, age, and race day temperature. A seperate model was built to predict each split time interval.

The first 5k split time was predicted using the previously mentioned attributes. The next split time, the 10k, was predicting using those attributes as well as the prediction from the 5k model. We continued using the prediction results in subsequent models to obtain more acurate predictions.
